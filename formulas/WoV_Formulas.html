Max Hit Formula

effectiveAttackLevel = attackLevel + 8
weaponDamageBonus = weaponDamage + 30
flatDamageBonus = power × 0.333333 × attackSpeed

maxHit = floor(5 + (effectiveAttackLevel × 1.1 × (weaponDamageBonus + flatDamageBonus)) / 50)
  • attackLevel — ATTACK level (melee) or ARCHERY level (bows), 1–99
  • weaponDamage — Sum of QUICK_DAMAGE + HEAVY_DAMAGE from the weapon
  • attackSpeed — Speed stat on the weapon (1.0–5.0)
  • power — Total MELEE_POWER or RANGE_POWER from all gear (boosted by gems)

Damage Range

minHit = floor(maxHit × 0.5)
avgHit = floor(maxHit × 0.75)
maxHit = maxHit

Every hit rolls uniformly between minHit and maxHit.

Gem Power Bonus

Gems in RING and AMULET slots boost only the matching stat on the item they're slotted into:

itemPower = floor(baseItemPower × (1 + gemValue / 100))

Weapon Types

  • Slash weapons — Use SLASH_ACCURACY vs enemy SLASH_DEFENSE
  • Pierce weapons (including bows) — Use PIECE_ACCURACY vs enemy PIECE_DEFENSE
  • Bows — Use ARCHERY level instead of ATTACK level
  • Two-handed (WEAPON_2H, BOW) — Disable the SHIELD slot
Accuracy Calculation

Roll Values

MaxAccuracyRoll = (attackLevel + 8) × (equippedAccuracy + 32)
MaxDefenceRoll = (defenceLevel + 8) × (equippedDefence + 16)

Constants: 8 = base level offset, 32 = base accuracy bonus, 16 = base defence bonus.

How Combat Rolls Work

Each attack, the game rolls a random number from 0 to MaxAccuracyRoll and another from 0 to MaxDefenceRoll. If the accuracy roll is strictly greater than the defence roll, the attack hits.

Hit Chance Formula

If MaxAccuracyRoll > MaxDefenceRoll:

hitChance = 1 − (MaxDefenceRoll + 2) / (2 × (MaxAccuracyRoll + 1))

Otherwise:

hitChance = MaxAccuracyRoll / (2 × MaxDefenceRoll + 1)

Hit chance is always between 0 and 1 (exclusive) — never exactly 0% or 100%.

Experience Calculation

XP Required for a Level

RequiredXP = 500 × (2^((Level - 1) / 5) - 1)
  • Level — The target level (1–99)
  • At Level 1, RequiredXP = 0
  • XP requirement doubles every 5 levels
  • Base scaling factor is 500

DPS & Kills Per Hour

DPS

DPS = (avgHit × hitChance) / attackInterval

Where attackInterval = attackSpeed × 0.6 seconds.

Time to Kill

secondsToKill = monsterHP / DPS

Kills Per Hour

KPH = 3600 / (secondsToKill + 15)

All monsters use a 15s respawn timer.
