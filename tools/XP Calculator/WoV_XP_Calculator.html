Overview

The WoV XP Calculator helps you plan your skilling journey in Winds of Valen. Select a skill, set your current and target levels, and instantly see how many actions (or crafting cycles) you need for each available recipe.

The calculator supports Mining, Smithing, Fishing, and Potion Making, with special supply chain calculations for Smithing and Potion Making.

XP Formula

Winds of Valen uses an exponential XP curve. The XP required for any given level is calculated as:

XP(level) = floor(100 × 1.1 ^ (level - 1))

The XP remaining to reach your target level is simply:

XP Remaining = XP(target) - XP(current)

The number of actions needed for a given recipe is:

Actions = ceil(XP Remaining / XP per action)

Include All Steps — Potion Making

When Include all steps is checked for Potion Making, the calculator shows only Bottle recipes but computes the total XP from the entire production chain:

Production Chain

  • Bottling — XP from bottling the final potion
  • Brewing — XP from brewing the base brew
  • Ingredient Preparation — XP from cutting, crushing, or reducing the non-essence ingredient
  • Essence Sourcing — If the brew requires more essence than the ingredient preparation produces, the deficit is sourced from Blue Gill (Cut → 2 Essence Glands, each Reduced → 10 Essence)

Example: Weak Health Potion

  • Bottle: 200 XP
  • Brew (Weak Health Brew): 100 XP, needs 10 Common Essence + 1 Common Trout
  • Cut Common Trout: 20 XP, yields 1 Cut Common Trout + 1 Essence Gland
  • Reduce Essence Gland: 10 XP, yields 10 Common Essence (covers the 10 needed)
  • Total: 330 XP/cycle

Small and Large bottle variants are consolidated since they yield the same net XP.

The materials column shows the total raw materials needed across all cycles.

Include All Steps — Smithing

When Include all steps is checked for Smithing, the calculator filters out Bars and shows only finished items (Armor, Weapons, Tools). The total XP includes:

  • Item Smithing — XP from smithing the final item
  • Bar Smelting — XP from smelting the required bars

Mining XP is not included since the calculator only counts XP for the selected skill.

Example: Bronze Platelegs (8 Bronze Bars)

  • Smith Bronze Platelegs: 120 XP
  • Smelt 8 Bronze Bars: 8 × 15 = 120 XP
  • Total: 240 XP/cycle
  • Raw materials: 8× Copper Ore, 8× Tin Ore

You can filter by sub-category (Armor, Weapons, Tools) while in all-steps mode.

Shareable URLs

The calculator encodes your current configuration into the URL as query parameters:

  • skill — The selected skill (mining, smithing, fishing, potion_making)
  • cat — The selected sub-category (omitted when "All")
  • cur / tgt — Current and target levels
  • allSteps — Whether "Include all steps" is enabled

Click the Share button to copy the URL to your clipboard. Anyone who opens the link will see the exact same configuration.

Data Sources

All recipe data is sourced from the community-maintained JSON files in this repository:

  • ores.json — Ore types, levels, and XP (entities)
  • bars.json, armor.json, weapons.json, tools.json — Smithing data embedded in entities
  • fish.json — Fish types, levels, XP, and cutting recipes (entities)
  • ingredients.json, potions.json — Potion making data embedded in entities

If you find inaccurate data, please submit a pull request or open an issue on GitHub.
