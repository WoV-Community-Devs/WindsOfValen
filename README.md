# WindsOfValen

A collection of tools, calculators, and data for WindsOfValen.

## Structure

- `data/` — Static game reference data (items, enemies, skills)
- `tools/` — Calculators and standalone tools
- `shared/` — Reusable utilities and formulas used across tools
- `docs/` — Guides, formula breakdowns, and methodology

## Tools

| Tool | Description |
|------|-------------|
| [XP Calculator](tools/xp_calculator/) | Calculate XP rates and time to level |
| [DPS Calculator](tools/dps_calculator/) | Calculate damage per second |
| [Money/Hour](tools/money_per_hour/) | Estimate gold/currency earn rates |

## Setup

```bash
pip install -r requirements.txt
```
